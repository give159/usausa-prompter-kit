# 自作ツールを仕事に使う 事例集（職種別・ひな形）

「単一ファイルのツールって、自分の仕事に使えるの？」に答える事例集です。受付・集計・チェックリストなど、**職種ごとの"あるある業務"を、小さなツールで楽にするアイデア**を、そのまま企画メモに使えるひな形つきでまとめました。プログラミングはできなくてOK。「こういうのが欲しい」を書ければ、作る人（またはAI）に頼めます。

## こんな人向け

- 自作ツール・自動化に興味はあるが、何に使えるか思いつかない人
- 毎日の手作業を減らしたいが、きっかけがない人
- 「うちの職種でも使える？」を確かめたい人
- 部署の業務改善のネタを探している人

## 単一ファイルツールが得意なこと

大がかりなシステムではなく「1つのファイルで動く小さな道具」が輝くのは、こんな場面です。

- 毎日・毎週くり返す、決まった手作業
- 紙やExcelで管理していて、転記ミスが起きやすい作業
- 人によってやり方がバラバラな作業（標準化したい）
- ネットにつなぎたくない・データを外に出したくない作業

逆に、大量データの本格処理や、複数人の同時作業は苦手です。**「小さく・手元で・くり返す」がキーワード**です。

## キットの中身

```
usecase-kit/
├─ README.md                     ← いまここ
├─ 00_使えるかの見分け方.md         ← 自分の業務が向いてるか判断する
├─ 01_事例_受付・窓口.md           ← 職種別アイデア
├─ 02_事例_集計・データ.md
├─ 03_事例_チェック・確認.md
├─ 04_事例_教育・研修.md
├─ 05_事例_営業・接客.md
└─ 06_企画メモ_穴埋めシート.md      ← 「作りたい」を1枚にまとめる
```

## 使い方

1. `00_見分け方` で、自分の業務が向いているか確かめる
2. 自分に近い職種の事例ファイル（`01`〜`05`）を読んで、アイデアを探す
3. ピンと来たら `06_企画メモ` を埋めて、作る人（またはAI）に渡す

> うさうさ研修工房の約束：**まず小さく始める。** 最初から完璧な大作を狙わず、「1つの手作業が楽になる」だけの小さな道具から。それが続けるコツです。
