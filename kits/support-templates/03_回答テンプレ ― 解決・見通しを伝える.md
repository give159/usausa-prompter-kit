# 03 回答テンプレ ― 解決・見通しを伝える

状況が分かったら、回答します。回答には3パターンあります。①解決した／②回避策がある／③まだ調査中。それぞれのテンプレを用意しました。

---

## パターン① 解決策を案内できるとき

```
＿＿＿様

お待たせしました。原因が分かりました。

＿＿＿（何が原因だったか・簡潔に）が原因でした。
お手数ですが、下記をお試しください。

1. ＿＿＿＿＿
2. ＿＿＿＿＿

これで解決するはずです。もし直らない場合は、
その旨をお知らせください。引き続き対応いたします。

よろしくお願いいたします。
＿＿＿（名前・役職名）
```

---

## パターン② すぐ直せないが、回避策があるとき

```
＿＿＿様

確認しました。＿＿＿（症状）については、現在＿＿＿（修正対応中／原因を調査中）です。

当面は、下記の方法で回避できます。
・＿＿＿＿＿

根本的な修正は＿＿＿（例：次回の更新／◯日ごろ）を予定しています。
完了しましたら改めてご連絡いたします。

ご不便をおかけしますが、よろしくお願いいたします。
＿＿＿（名前・役職名）
```

---

## パターン③ まだ原因が分からないとき（正直に）

```
＿＿＿様

ご報告いただいた件、確認を進めていますが、
現時点では原因の特定に至っておりません。

引き続き調査し、＿＿＿（例：◯営業日以内）を目安に、
分かったことをご連絡いたします。

お時間をいただき恐れ入りますが、よろしくお願いいたします。
＿＿＿（名前・役職名）
```

> 【約束】③は「逃げ」ではありません。分からないことを正直に、次の連絡日つきで伝えるのは、誠実な対応です。憶測で「直りました」と言うより、ずっと信頼されます。

---

## 回答のあとに1つだけ

解決したら、可能なら**再発防止**を一言添えると質が上がります。
例:「今回の件を受けて、次の更新で保存の仕組みを見直します」
――ただし、できない約束はしないこと。言った以上はやる、が信頼のもとです。
