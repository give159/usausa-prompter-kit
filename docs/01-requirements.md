## 1. 目的と背景

勉強会・LT・セミナー・配信・動画収録において、登壇者が「緊張で原稿を見失う」「時間が読めない」「原稿を見ると下を向いてしまう」という課題を抱えている。本ツールは、原稿を設定した速度で自動スクロール表示するテレプロンプターとして、前を向いたまま・時間どおりに話せる状態を支援する。

## 2. 提供形態・利用者

- **提供形態**: 単一ファイルHTML（インストール不要、ブラウザで開くだけ）
- **動作環境**: 2系統
  - 通常版: ローカルで直接開く（file:// またはローカルサーバ）。AI相談は無効
  - アーティファクト版: Claudeのアーティファクト環境で実行。AI相談が有効
- **想定利用者**: 勉強会・LT登壇者、研修講師、配信者、動画制作者（非エンジニアを含む）

## 3. 用語

| 用語 | 定義 |
|---|---|
| CPM | 読み上げ速度（字/分・Characters Per Minute） |
| しおり | 中断位置を記憶し次回そこから再開する機能 |
| 台本ボックス | 原稿を2枠まで保存できるスロット |
| 勉強ノート | ツール内に保存できるメモ（kind: study） |
| アーティファクト版 | Claude上でHTMLを実行する環境。AI相談が動作する |

## 4. 機能要件（FR）

### 4.1 表示・再生系

| ID | 要件 | 仕様値 |
|---|---|---|
| FR-01 | 読み上げ速度の調整 | 80〜600 字/分（10刻み、既定300） |
| FR-02 | 文字サイズの調整 | 20〜120 px（既定48） |
| FR-03 | 表示方向の切り替え | 縦スクロール（横書き・下→上）／縦書き（右→左）／縦書き（左→右） |
| FR-04 | ミラー表示 | 画面左右反転（プロンプター機材の反射投影用） |
| FR-05 | 配色カスタム | 文字色・背景色を選択可。WCAGコントラスト比で読みやすさを自動判定（4.5以上=読みやすい／3以上=ぎりぎり／3未満=読みにくい） |
| FR-06 | カウントダウン | 再生前 0〜10 秒（既定3秒）、カウント音のオン/オフ可 |
| FR-07 | タイマー | 1〜180 分（既定30分） |

### 4.2 原稿・データ系

| ID | 要件 | 仕様値 |
|---|---|---|
| FR-11 | 原稿の自動保存 | 各操作の都度＋画面を閉じる時にブラウザへ保存 |
| FR-12 | 版履歴 | 最大12件（超過分は最古から自動削除） |
| FR-13 | 読み上げ実績の記録 | 最大50件（予定/実測時間。超過分は最古から自動削除） |
| FR-14 | 文字数ガード | 20,000字で任意警告、50,000字で強い警告 |
| FR-15 | 原稿の取り込み | .docx／.xlsx／.csv／.txt／.md／.json |
| FR-16 | バックアップ | JSON形式で書き出し／読み込み |
| FR-17 | 台本ボックス | 固定2枠（名称は40文字まで） |
| FR-18 | 勉強ノート | メモの保存・編集・削除 |

### 4.3 操作系

| ID | 要件 | 仕様値 |
|---|---|---|
| FR-31 | しおり | 位置の記憶と「続きから」再開 |
| FR-32 | キーボードショートカット | Space=再生/一時停止、←/→=少し戻す/進める、↑/↓=速度±10、R=最初から、B=しおり、M=ミラー |
| FR-33 | スリープ防止 | Screen Wake Lock API による画面消灯防止（オン/オフ可） |

### 4.4 AI連携系（アーティファクト版でのみ有効）

| ID | 要件 | 仕様値 |
|---|---|---|
| FR-19 | AI相談 | チャット形式の相談。APIキー入力は不要（認証は実行環境に委譲） |
| FR-20 | AIペルソナ | 3種（うさうさアナウンサー／うさうさ監督／うさうさマネージャー） |
| FR-21 | クイックアクション | 台本を整える／概要欄要約／オープニング挨拶／読み方のコツ |
| FR-22 | 通常版での挙動 | 案内表示のみ（「AI相談は使えません」）。通信は発生しない |

## 5. 非機能要件（NFR）

| ID | 区分 | 要件 |
|---|---|---|
| NFR-01 | 可用性 | AI連携系（FR-19〜21）を除く全機能はオフラインで完結 |
| NFR-02 | 構成 | 単一ファイルHTML。外部script・CSS・CDN読み込みゼロ |
| NFR-03 | データ保護 | 保存先は端末内ブラウザ領域（localStorage）のみ。AI相談使用時を除き外部送信なし |
| NFR-04 | 対応ブラウザ | Chrome 84+／Firefox 126+／Safari 16.4+（DecompressionStream・Web Audio・Wake Lock対応が目安） |
| NFR-05 | 取り込み制約 | Office系は.docx/.xlsx（ZIP+XML形式）のみ。2007年より前の.doc/.xls（バイナリ形式）は非対応 |
| NFR-06 | 復元安全性 | 保存データ復元時は全フィールドを型チェック・clampし、不正値は既定値へ |

## 6. 制限値一覧（コード定数・実測確認済み）

| 定数 | 値 |
|---|---|
| 読み上げ速度（cpm） | 80〜600（step 10） |
| 文字サイズ | 20〜120 px |
| タイマー | 1〜180 分 |
| カウントダウン | 0〜10 秒 |
| 版履歴上限 | 12 件 |
| 実績ログ上限 | 50 件 |
| 文字数警告 | 20,000／50,000 字 |
| 台本ボックス | 2枠・名称40文字 |
| AI応答上限 | max_tokens 1000（model: claude-sonnet-4-6） |

## 7. スコープ外

- 複数人での同時編集、クラウド同期
- 音声認識による自動追従
- .doc/.xls（旧バイナリ形式）の取り込み
