# 付録4｜利用手順書（お客様向け） ― うさうさプロンプター

対象: v1.7B PRO ／ むずかしい知識は不要です。この1枚で今日から使えます。

---

## 1. うさうさプロンプターとは

発表用の原稿を、ちょうどいい速さで自動的に流してくれる「電子カンペ」です。前を向いたまま話せるので、勉強会・研修・オンライン配信・動画収録で活躍します。インストールは不要、ファイルをブラウザで開くだけです。

## 2. かんたん3ステップ

1. **開く** — 配布された `usausa-prompter-v1_7B-PRO.html` をダブルクリック（ChromeやEdge等のブラウザで開きます）
2. **貼る** — 原稿の入力欄に、発表原稿を貼り付けます（WordやExcelのファイルからの取り込みもできます）
3. **再生** — 再生ボタンを押すと、3秒のカウントダウンのあと原稿が流れ始めます

速さが合わないときは、速度（字/分）を上下させてください。80〜600の間で調整でき、最初は300が目安です。文字が小さければ文字サイズ（20〜120）を大きくします。

## 3. 主な機能

| 機能 | できること |
|---|---|
| しおり | 中断した場所を記憶。「続きから」ボタンで再開できます |
| タイマー | 発表時間（1〜180分）を設定し、時間どおりの進行を支援します |
| 読み上げ実績 | 予定時間と実際にかかった時間を記録（最新50件）。練習の振り返りに |
| 取り込み | Word（.docx）・Excel（.xlsx）・テキスト・CSV・Markdown・JSONから原稿を読み込めます |
| 台本ボックス | よく使う原稿を2枠まで保存できます |
| 表示方向 | 横書き（下から上へ流れる）／縦書き（右→左）／縦書き（左→右） |
| ミラー表示 | 画面を左右反転。プロンプター機材に映すときに使います |
| 配色 | 文字色と背景色を選べます。読みやすさは自動で判定して教えてくれます |
| バックアップ | 設定と原稿をJSONファイルに書き出し・読み込みできます |

## 4. キーボードショートカット

| キー | できること |
|---|---|
| Space | 再生 ⇔ 一時停止 |
| ← / → | 少し戻す / 少し進める |
| ↑ / ↓ | 速度を10ずつ変える |
| R | 最初からリセット |
| B | しおりを付ける |
| M | ミラー表示を切り替え |

## 5. データの保存と安全について

- 原稿と設定は、お使いの端末のブラウザの中にだけ保存されます。外部への送信はありません
- 保存は自動です。ブラウザを閉じても、次に開くと続きから使えます
- 共有パソコンでお使いの場合は、使用後に原稿を消すことをおすすめします（ブラウザのデータ削除で消えます）
- 「相談」ボタン（AI機能）は、Claudeのアーティファクト版で開いたときだけ動きます。お手元のファイルを直接開く通常版では「AI相談は使えません」と表示されます。

## 6. 困ったときは

| 症状 | 対処 |
|---|---|
| 保存されない | ブラウザのプライベート（シークレット）モードになっていないか確認してください |
| WordやExcelが取り込めない | 古い形式（.doc/.xls）は非対応です。一度開いて .docx/.xlsx で保存し直してから取り込んでください |
| 速度が合わない | ↑↓キーで10ずつ微調整できます。実績記録の「予定と実測の差」も参考になります |
| 相談ボタンが使えない | 通常版（ローカルで開いた場合）ではAI相談は動きません。仕様どおりの動作です |
| それでも解決しない | 虫の絵のボタンを押すと「状況メモ」がコピーできます。それを添えて配布元へご相談ください |

## 7. 動作環境

Chrome 84以降／Firefox 126以降／Safari 16.4以降を目安にしてください。インターネット接続は不要です（AI相談を除く）。
